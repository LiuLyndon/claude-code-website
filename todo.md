
  GitHub Actions 觸發條件                                                                                                                                  
  - 每次 push 到 master 分支時自動執行                                                                                                                    
  執行步驟                                                                                                                                                   1. 安裝 Node.js 20 + npm ci                                                                                                                                2. 執行 ng build --base-href /claude-code-website/（對應 GitHub Pages 路徑）                                                                               3. 將 dist/claude-code-website/browser/ 複製到 /docs                                                                                                       4. 自動 commit & push /docs 回 master                                                                                                                                                                                                                                                                                 GitHub Pages 設定確認                                                                                                                                      - Source: master 分支 → /docs 資料夾 ✓                                                                                                                     - 網站路徑: https://liulyndon.github.io/claude-code-website/ ✓                                                                                                                                                                                                                                                      
  Push 完成後約 1-2 分鐘，Actions 會自動跑起來，可在 https://github.com/LiuLyndon/claude-code-website/actions 查看進度。
